# Pollastres – Anàlisi del comportament mitjançant seguiment de keypoints

> **Analisis del comportament dels pollastres fent seguiment dels keypoints**

This project provides tools to **track chicken body keypoints** across video
frames and to **classify the resulting trajectories into behavioural states**
(inactive, locomotion, foraging, pecking, preening).

---

## Repository layout

```
pollastres/

```

## Keypoints tracked

Three body-part keypoints are tracked for each chicken:

| ID | Keypoint | Description                  |
|----|----------|------------------------------|
| 0  | `tail`   | Base of the tail             |
| 1  | `neck`   | Neck / body centre           |
| 2  | `head`   | Top of the head / beak area  |

---

## Behavioural states

| State        | Description                             |
|--------------|-----------------------------------------|
| `inactive`   | Resting or standing still               |
| `locomotion` | Walking or running                      |
| `foraging`   | Searching for food (slow body movement) |
| `pecking`    | Eating or interacting with the ground   |
| `preening`   | Grooming / cleaning feathers            |

---

## Features extracted from keypoint trajectories

| Feature           | Description                                              |
|-------------------|----------------------------------------------------------|
| `vel_body`        | Body (neck) speed frame-to-frame                         |
| `vel_head`        | Head speed frame-to-frame                                |
| `acc_head`        | Head acceleration                                        |
| `head_length`     | Magnitude of neck→head vector                            |
| `parallel`        | Head movement component along body axis                  |
| `perpendicular`   | Head movement component perpendicular to body axis       |
| `dist_body_20`    | Total body movement in a ~2 s rolling window             |
| `dist_body_50`    | Total body movement in a ~5 s rolling window             |
| `angle_var`       | Variance of head–body angle in a ~2 s rolling window     |
| `ratio_head_body` | Ratio of head speed to body speed                        |

## Python pipeline (`analisi_keypoints.py`)

### Requirements

```bash
pip install -r comportament/requirements.txt
```

### Inputs

The script expects a CSV file with (at minimum) the following columns:

| Column   | Description                          |
|----------|--------------------------------------|
| `frame`  | Frame index (integer, 1-based)       |
| `state`  | Ground-truth behaviour label         |
| `tail_x` | Tail keypoint – x coordinate         |
| `tail_y` | Tail keypoint – y coordinate         |
| `neck_x` | Neck keypoint – x coordinate         |
| `neck_y` | Neck keypoint – y coordinate         |
| `head_x` | Head keypoint – x coordinate         |
| `head_y` | Head keypoint – y coordinate         |

If no CSV is found, the script generates a synthetic dataset automatically.

### Running

```bash
# Use simulated data generated by the R script
cd comportament/scripts
python analisi_keypoints.py

# Or point to your own tracked-keypoint CSV
CHICKEN_CSV=/path/to/keypoints.csv python analisi_keypoints.py

# Override video parameters
CHICKEN_FPS=25 CHICKEN_CSV=/path/to/keypoints.csv python analisi_keypoints.py
```

### Outputs

| File                          | Description                                                |
|-------------------------------|------------------------------------------------------------|
| `behaviour_time_budget.png`   | Bar chart of estimated minutes per behaviour               |
| `keypoint_trajectories.png`   | Neck trajectory coloured by predicted behaviour            |
| `keypoint_snapshots.png`      | Grid of skeleton snapshots at evenly spaced time-points    |
| `confusion_matrix.png`        | Classifier confusion matrix (requires ground-truth labels) |

## R script (`SimulaMovimiento1pollo_ClassificaYCuantificaComportamientos.R`)

The R script generates **labelled synthetic keypoint data** for a single
chicken using a Markov-chain behaviour model, trains a Random Forest
classifier on those labels, and saves:

- `chicken_simulated_data_full.csv` – raw positions + behaviour labels
- `chicken_features_for_classification.csv` – normalised feature matrix
- `behaviour_time_budget.png` – time-budget bar chart

Run it with:

```r
Rscript comportament/scripts/SimulaMovimiento1pollo_ClassificaYCuantificaComportamientos.R
```

The generated `chicken_simulated_data_full.csv` can then be consumed directly
by `analisi_keypoints.py`.

## Extending to real video data

To apply this pipeline to real video recordings:

1. Run a pose estimator (e.g. [DeepLabCut](https://www.mackenziemathislab.org/deeplabcut),
   [SLEAP](https://sleap.ai/), or [YOLOv8-pose](https://docs.ultralytics.com/tasks/pose/))
   on your footage to obtain per-frame keypoint coordinates.
2. Export the predictions to a CSV that matches the column schema above
   (use `state` = `"unknown"` or omit it if no ground-truth labels exist).
3. Run `analisi_keypoints.py` with `CHICKEN_CSV` pointing to your file.
